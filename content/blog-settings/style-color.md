## Markdown 样式变量

Markdown文章中的所有样式变量都在`app\assets\styles\markdown-core.css` 当中。<br>
推荐在 `app\assets\styles\themes\custom.css` 中找到对应值并修改，同时在主页切换为**自定义**主题

### 文本颜色变量

| 变量名 | 说明 | 使用场景 |
| --- | --- | --- |
| `--md-text-color` | 正文文本颜色 | 段落、列表项等主要文本内容 |

### 标题颜色变量

| 变量名 | 说明 | 对应元素 |
| --- | --- | --- |
| `--md-h1-color` | 一级标题颜色 | `# 标题` |
| `--md-h2-color` | 二级标题颜色 | `## 标题` |
| `--md-h3-color` | 三级标题颜色 | `### 标题` |
| `--md-h4-color` | 四级标题颜色 | `#### 标题` |
| `--md-h5-color` | 五级标题颜色 | `##### 标题` |
| `--md-h6-color` | 六级标题颜色 | `###### 标题` |

### 边框颜色变量

| 变量名 | 说明 | 使用场景 |
| --- | --- | --- |
| `--md-border-color` | 通用边框颜色 | H2 下划线、表格边框、水平分隔线 |

### 引用块样式变量

| 变量名 | 说明 | 使用场景 |
| --- | --- | --- |
| `--md-blockquote-border` | 引用块左边框颜色 | `> 引用内容` 的左侧边框 |
| `--md-blockquote-color` | 引用块文本颜色 | `> 引用内容` 的文字颜色 |
| `--md-blockquote-bg` | 引用块背景颜色 | `> 引用内容` 的背景 |

### 代码样式变量

| 变量名 | 说明 | 使用场景 |
| --- | --- | --- |
| `--md-inline-code-bg` | 行内代码背景颜色 | \`代码\` |
| `--md-inline-code-color` | 行内代码文本颜色 | \`代码\` 的文字颜色 |
| `--md-code-bg` | 代码块背景颜色 | \`\`\`代码块\`\`\` 的背景 |
| `--md-code-color` | 代码块文本颜色 | \`\`\`代码块\`\`\` 的文字颜色 |

### 表格样式变量

| 变量名 | 说明 | 使用场景 |
| --- | --- | --- |
| `--md-table-header-bg` | 表头背景颜色 | Markdown 表格的表头单元格 |

### 链接样式变量

| 变量名 | 说明 | 使用场景 |
| --- | --- | --- |
| `--md-link-color` | 链接文本颜色 | `[链接文本](url)` |

### 文本强调变量

| 变量名 | 说明 | 使用场景 |
| --- | --- | --- |
| `--md-strong-color` | 粗体文本颜色 | `**粗体**` 或 `__粗体__` |
| `--md-em-color` | 斜体文本颜色 | `*斜体*` 或 `_斜体_` |
