# Markdown 功能概览

本页展示博客支持的所有自定义 Markdown 语法。每个功能均提供**语法示例**与**实际效果**对照。

---

## Callout 提示框

提示框使用 Obsidian 兼容的 `> [!type] 标题` 语法，内部支持完整的 Markdown 渲染（加粗、链接、代码等）。

### info — 信息

**语法：**

```
> [!info] 这是标题
> 这里是内容，支持 **加粗**、*斜体* 和 `行内代码`。
```

**效果：**

>[!info] 这是标题
> 这里是内容，支持 **加粗**、*斜体* 和 `行内代码`。

---

### note — 笔记

**语法：**

```
> [!note] 笔记
```

>[!note] 笔记
> 适合记录补充信息或旁注内容。

### 多行内容 & 嵌套 Markdown

Callout 内部支持多段内容，包括列表、链接等。

**语法：**

```
> [!info] 多行示例
> 第一段内容，支持 **加粗** 和 [链接](https://example.com)。
>
> 第二段内容，空行会被保留。
>
> - 列表项 1
> - 列表项 2
```

**效果：**

>[!info] 多行示例
> 第一段内容，支持 **加粗** 和 [链接](https://example.com)。
>
> 第二段内容，空行会被保留。
>
> - 列表项 1
> - 列表项 2

## 代码块（含复制按钮）

所有代码块右上角自动添加 **Copy** 按钮，点击后将纯文本内容复制到剪贴板。

**语法：**

````
```typescript
function greet(name: string): string {
  return `Hello, ${name}!`
}

console.log(greet('World'))
```
````

**效果：**

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`
}

console.log(greet('World'))
```

---

支持语言高亮：`ts` / `js` / `vue` / `json` / `bash` / `html` / `css` / `python`。

**语法：**

````
```bash
npm install @nuxt/content
nuxt dev
```
````

**效果：**

```bash
npm install @nuxt/content
nuxt dev
```

## 数学公式（KaTeX）

使用 LaTeX 语法书写数学公式。行内公式用 `$...$`，块级公式用 `$$...$$`。

### 行内公式

**语法：**

```
爱因斯坦质能方程：$E = mc^2$
```

**效果：**

爱因斯坦质能方程：$E = mc^2$

---

### 块级公式

**语法：**

```
$$
\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}
$$
```

**效果：**

$$
\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}
$$

---

**语法：**

```
$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
$$
```

**效果：**

$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
$$

## 标准 Markdown

以下为标准 Markdown 语法，已通过样式美化。

### 标题层级

```
# H1 一级标题
## H2 二级标题
### H3 三级标题
#### H4 四级标题
```

### 文字样式

```
**加粗文字**
*斜体文字*
~~删除线~~
`行内代码`
```

**效果：** **加粗文字** / *斜体文字* / ~~删除线~~ / `行内代码`

### 引用

**语法：**

```
> 这是一段引用文字。
> 可以跨越多行。
```

**效果：**

> 这是一段引用文字。
> 可以跨越多行。
